About the Bisection Method

The bisection method is a root-finding algorithm that repeatedly bisects an interval and then selects a subinterval in which a root must lie for further processing.

How it works:

  1. Start with an interval [a, b] where f(a) and f(b) have opposite signs.
  2. Calculate the midpoint c = (a + b) / 2.
  3. Evaluate f(c).
  4. If f(c) = 0, c is a root. If not, replace either (a, f(a)) or (b, f(b)) with (c, f(c)), so that there is a sign change between the two points.
  5. Repeat steps 2-4 until the interval is sufficiently small or the maximum number of iterations is reached.

Formula:

\[c_n = \frac{a_n + b_n}{2}\] where \(c_n\) is the midpoint at iteration n, and \(a_n\) and \(b_n\) are the interval endpoints.

Convergence Rate:

The bisection method has a linear convergence rate. The error is approximately halved in each iteration:

\[|x - c_n| \leq \frac{b - a}{2^{n+1}}\] where x is the true root, \(c_n\) is the nth approximation, and [a, b] is the initial interval.

Advantages:

  • Simple to understand and implement
  • Guaranteed to converge for continuous functions
  • Robust method that always works when the initial conditions are met

Disadvantages:

  • Relatively slow convergence compared to some other methods
  • Requires that the function changes sign over the given interval
  • May not find the root if the function only touches the x-axis without crossing it
